# Projet Hardhat – Tontine DApp (Base Technique)

Ce dépôt contient la **mise en place complète de l’environnement Hardhat** pour le développement de smart contracts Ethereum.  
Il constitue la **base technique** d’une future application de **tontine décentralisée**.

Le travail réalisé couvre :
- l’installation et la configuration de Hardhat
- la création de la structure du projet
- l’écriture des premiers smart contracts
- la compilation et les tests de base

---

##  Objectif de cette étape

- Mettre en place un **environnement de développement blockchain fonctionnel**
- Comprendre le workflow Hardhat (compile, test, run)
- Créer un **premier smart contract ERC-20** servant de base aux cotisations d’une tontine
- Préparer le projet pour les prochaines étapes (gestion de tontine, déploiement testnet)

---

##  Technologies utilisées

- **Node.js**
- **npm**
- **Hardhat v3**
- **Solidity 0.8.28**
- **OpenZeppelin Contracts**
- **TypeScript / JavaScript**
